# Buscador de CEP

## Descricao
API para busca e consulta de enderecos por CEP (Codigo de Enderecamento Postal).

## Funcionalidades
- Consulta de CEP
- Busca por endereco
- Validacao de CEP
- API REST

## Tecnologias
- Java
- Spring Boot
- ViaCEP API

## Uso
`ash
GET /api/cep/{cep}
`

## Instalacao
`ash
git clone https://github.com/Wsanbey/Buscador-de-CEP.git
cd Buscador-de-CEP
mvn install
`
